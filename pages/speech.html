常用术语

狼人杀是一款主要靠发言、推理和投票推进的游戏，对局中经常会出现许多专用说法。新手刚上桌时，可能听完一大段发言仍不明白对方在说什么。这里整理了面杀和网杀中常见的狼人杀术语，帮助你快速听懂发言、看懂套路；部分说法会随地区、版型和房规略有变化。

闭眼玩家
夜晚未能睁眼采取行动的身份。
民及民以上
身份至少是平民，也可能是神职，意思是暂时不把该玩家放进狼坑。
板子
该局神职、平民、狼人各自的人数、技能与规则配置。
配置
形容一名玩家的游戏能力。
人设
“人物设定”的简称，指一名玩家平时的游戏风格。
狼杀
意同“刀”，指夜里被狼人杀死。除第一晚外，被狼杀的玩家通常不能发表遗言。
自刀
狼人选择杀死自己，常用于骗取女巫解药和好人信任。
空刀
狼人夜里选择不杀人，该晚即为空刀。
毒杀
夜里被女巫毒死。死者通常没有遗言，猎人或狼王也不能发动枪杀技能。
枪杀
意同“带”，指猎人或狼王出局时开枪带走一名玩家。
压枪
猎人死亡时选择不开枪。
闷枪
猎人或狼王因中毒、狼美人魅惑或情侣殉情等原因出局，无法发动枪杀技能。
殉情
狼美人出局时带走前一晚魅惑的玩家；也指情侣一方出局后，另一方随之出局。殉情的猎人或狼王不能发动技能。
倒牌
玩家死亡或被放逐出局；有些地区主要用来指夜间死亡。
银水
被女巫使用解药救活的玩家。
金水
预言家查验出的好人。口语中有时也泛指场上已经确定的好人；“骑金”指骑士验证出的好人，“商金”指黑市商人交易成功的好人。
金银花露水
既被女巫救活，又被预言家查验为好人的玩家，身份通常很高。
金刚狼
底牌是狼人，却被场上多数玩家当作铁好人的高身份狼人；金水、银水或成功穿衣服都可能帮助其形成身份。
查杀
预言家查验出的狼人；也可指被骑士验证为狼人，或黑市商人交易失败的玩家。
开毒
女巫使用毒药技能。
盲毒
女巫首夜被杀且无法自救，在未听任何发言前使用毒药。
自证身份
通过女巫开毒、猎人开枪、骑士明验等技能证明自己的身份。
视野回推
意为等场上出现更多信息后再重新盘点，常见于前置位暂时缺少信息时。
做好 / 坐好
一名玩家在发言、逻辑或投票上，被认为更可能是好人。
表水
意同“发言”；也指玩家被质疑时，通过发言说明自己为什么是好人。
爆水
好人在发言中表现出明显的平民或好人视角。
铁狼 / 定狼
认为某名玩家确定是狼人。
狼坑
发言时列出自己怀疑为狼人的玩家号码。
抿身份 / 抿牌
开局确认身份牌后，通过观察其他玩家的表情与反应猜测身份。
卦象
把玩家看牌后的表情和肢体动作当作判断信息。
夜卦
观察玩家夜间的表情、动作，或比较闭眼前后的动作变化。
视角
一名玩家对场上其他玩家和局势信息的了解范围。
炸身份 / 诈身份
通过假跳神职、假报查杀、金水或银水来观察目标反应，试探其身份。使用后要及时说明并讲清退水原因。
丢水包
首日前置位对后置位施加发言压力，制造可供讨论的话题。
场外
与发言、表情和投票等游戏内容无关的因素，例如夜间动静。
焦点牌
因发言怪异或被多人谈及，而受到高度关注的玩家。
聊爆
发言出现逻辑漏洞或不该拥有的开阔视角，暴露出狼人可能性。
抗推
好人在白天被当作狼人放逐，相当于替狼人承担了一次白天轮次；也可指某名玩家成为当天主要放逐目标。
生推
没有预言家查杀时，主要依靠发言和票型投票放逐。
操作
指特定战术安排，例如“打这波操作”。
收益
一项选择给某个阵营带来的好处，例如“对好人的收益”或“对狼人的收益”。
划水
发言过于简略，快速带过且不主动寻找狼人。
黄金划水位
首轮第一位发言的玩家，因为信息最少，发言通常较短。
单边预
第一天只有一名玩家起跳预言家。
反水
不承认给自己发金水的人是真预言家。
接金水
被预言家或通灵师发金水后，承认并认可对方给出的验人结果。
反水立警
被发金水的玩家起跳预言家，与发金水者对立。
反向金水
悍跳狼给一名玩家发查杀，悍跳狼出局后，被查杀者因反向逻辑身份升高。
跟风
发言只跟随其他玩家，没有提供自己的判断和依据。
贴脸
使用与逻辑无关的情绪、人格担保或极端誓言证明身份。
站边
在两个对立判断或阵营中，选择支持其中一方。
共边 / 认识
指两名玩家被判断为同一阵营；“不共边 / 不认识”则指不同阵营。站边多用于第一人称，共边多用于描述他人关系。
软站边
只在最低限度上勉强认可某一方，态度仍有保留。
污 / 脏
把一名玩家的身份做低，使其更容易被怀疑。
盘
梳理场上信息并进行推理。
踩
指出一名玩家发言或状态不好的地方，并怀疑其可能为狼人。
裸点
第一天信息很少时，直接点出自己认为像狼人的玩家。
容错率
表示狼坑中的某名玩家相对不那么像狼，可以暂时保留。
轮次
双方阵营在当前局面的领先程度，也指本轮优先考虑放逐的范围。
警推在先
首轮出局的是狼人，好人轮次领先。
狼推在先
首轮出局的是好人，狼人轮次领先。
屠边 / 屠神 / 屠民
狼人集中淘汰全部神职或全部平民，以满足屠边胜利条件。
屠城
狼人淘汰所有好人。
跳 / 认
意同“拍身份”，公开宣称自己是某个神职或平民，例如“跳预言家”“认猎人”。
原地起跳
被发查杀的玩家当场起跳预言家，通常会受到更严格的判断。
假跳
不是某身份的玩家，自称是该神职或平民。
悍跳
狼人假跳神职，最常见的是假跳预言家。
滴滴代跳
好人替预言家或通灵师起跳预言家或通灵师。
上黑车
狼人替预言家或通灵师起跳预言家或通灵师。
对跳
两名或以上玩家同时宣称自己是同一个神职。
罗汉跳
两名狼人同时起跳预言家。
挡刀
好人假跳某个神职，降低真正神职被狼人杀死的风险。
穿衣服
假跳某个身份，通常特指假跳神职；好人和狼人都可能穿衣服。
脱衣服
假跳神职后，再说明自己并非该神职。
退水
承认自己是假跳身份；在警长竞选中也指退出竞选。
公共狼坑
全场身份最低、甚至被狼队友一起踩的玩家。
认狼
公开承认自己的狼人身份。
狼互咬 / 狼咬狼
两名狼人刻意互踩、互相攻击，让好人误以为二人不是同伴。
冲锋狼
发言积极、明显站边悍跳狼，并为悍跳狼争取票数的狼人。
倒钩 / 倒钩狼
狼人为了隐藏身份，故意支持好人阵营或攻击狼队友。
深水 / 深水狼
指身份隐藏很深、长期不在焦点位或被多数人认作好人的狼人。
摸头金
悍跳狼给好人发金水，尝试拉取该玩家的信任和票。
狼金水
悍跳狼给狼队友发金水，用于做高同伴身份。
怂狼局
狼人阵营没有人悍跳预言家的对局。
自爆
狼人主动公开身份并出局，立即终止当前白天发言并进入黑夜；是否拥有指刀权、白狼王能否带人，以当局房规为准。
指刀 / 点刀
狼人向队友明确夜里要击杀的目标；部分房规允许白天自爆的狼人指定当晚刀口。
拍刀
狼队公开底牌并宣布之后的击杀顺序；若好人即使掌握全部信息仍无法反击，则直接判定狼队获胜。
交牌
狼队判断已无法获胜，选择投降并提前结束游戏。
上票
意同“投票”。
票死 / 投死
意同“放逐”。白天投票中得票最高的玩家被淘汰，并按房规发表遗言。
平票
投票结束后，有至少两名玩家并列最高票。
辩论
平票后由最高票玩家分别追加发言；若再次投票仍平票，通常无人出局并直接入夜。
PK / PK 台
平票玩家进入额外发言和再次投票的环节。“PK”多作动词，“PK 台”多作名词。
归票
由存活的末置位玩家或警长总结狼坑，尽量统一票数放逐一名玩家。
绑票 / 冲票
几名玩家有计划地把票集中投给同一人。“绑票”强调事先统一，“冲票”常指临近投票时集中票数抢轮次。
跟票
观察其他玩家是否投票，或看清票型后才进行投票。
弃票
该轮没有投票。好人或狼人都可能弃票。
票型
记录每名玩家把票投给了谁，用于判断阵营和团队关系。
警下
没有参加警长竞选的玩家，通常拥有警长竞选投票权。
上警
第一天白天举手参加警长竞选、争夺警徽的行为。上警玩家通常没有警长竞选投票权。
警上
已经参加警长竞选的玩家。
警徽流
预言家提前说明后续夜晚的查验顺序，以及自己死亡后如何通过移交或撕毁警徽传递验人结果；标准局通常会安排两夜。
撕警徽
警长出局时选择不移交警徽。部分地区也会把投票放逐警长称为“撕警徽”。
吞警徽
警长竞选发言阶段有狼人自爆，竞选被打断，本局没有产生警徽；也可泛指通过自爆让既定警徽流失效。
真金水
已经确认的真预言家查验出的好人。
双金水
两名对跳预言家都给同一名玩家发金水。在普通对跳结构中该玩家通常身份很高，但特殊版型仍需结合规则判断。
双查杀
两名对跳预言家都给同一名玩家发查杀。在普通对跳结构中该玩家通常进入公共狼坑，但仍要防特殊战术和版型影响。
铜水
守卫守中了狼人当晚的刀口而形成的平安夜目标；只有在守护信息能够确认时才有参考价值。
金银双水
同一名玩家既是预言家的金水，又是女巫救过的银水，也常被称为“金银花露水”。
遗言
玩家出局后，按规则获得的最后一次发言机会。
前置位
同一轮发言中相对较早发言的位置。
后置位
同一轮发言中相对较晚发言的位置。
末置位 / 归票位
同一轮最后发言的位置，通常负责总结信息并给出归票建议。
失联 / 黑麦 / 断麦（段麦）
网杀中玩家暂时不在线，或其语音无法被其他玩家听见。
高贵
形容身份较高，通常暗指神职牌。
挂相
看牌、听发言时出现的面部表情和细微动作。挂相只能作为辅助信息，不能代替发言、票型和规则逻辑。
状态
玩家在表情、语气、发言节奏和动作上呈现出的整体感觉，也可与其平时表现作比较。
心态
一段发言背后的心理动机，例如是在找狼、保人、推人，还是为某种结果提前铺路。
做规矩
用投票惩罚明显违反房规或严重扰乱正常对局的玩法。是否执行应由全桌事先约定，避免把个人分歧当作规则。
首刀
狼人第一夜选择的击杀目标。
首验
预言家第一夜选择的查验目标。
首刀保护
连续多局游戏时，上一局首夜被刀的玩家下一局不能再次成为首刀目标的房规。
首验保护
连续多局游戏时，上一局首夜被验的玩家下一局不能再次成为首验目标的房规。
续刀
前一夜某个目标因被救或被守没有死亡，狼人下一夜继续选择击杀该玩家。
刀法
狼队选择夜间击杀目标的思路和顺序。
交刀
狼队判断已经无法获胜，选择认输并提前结束游戏，也常称“交牌”。
平安夜
一夜结束后没有玩家死亡，可能由空刀、守护、解药或其他技能造成。
空守
守卫当晚不守护任何玩家；是否允许空守以房规为准。
同守同救
狼人刀中守卫守护的玩家，同时女巫又对该玩家使用解药。该玩家是否死亡取决于当局规则，常见规则为同守同救死亡。
骗药
通过自刀、假信息等战术诱导女巫错误使用解药或毒药。
撒毒
女巫在夜间使用毒药，也称“开毒”。
可乐
女巫毒药的口语称呼。
验尸官
预言家的查验目标当夜刚好死亡，导致验人信息难以继续发挥白天带队价值。
分票
同一阵营没有把票集中给同一目标，而是分散投给多人。
拉票
通过发言争取其他玩家把票投向自己支持的目标，或在警长竞选中争取警下票。
压票
利用发言或票数优势压住另一方的票，使其无法形成有效冲票。
放票
该轮选择弃票或不跟随主要归票目标。
查票
查看或复核本轮票型，确认每名玩家把票投给了谁。
深推
预言家等主要信息位出局、场上缺少明确查验时，依靠发言、关系和票型继续放逐。
捞
发言替一名被怀疑的玩家解释，试图避免其被放逐；也可指女巫夜里使用解药救人。
裸打
不再隐藏身份或战术意图，直接亮明身份推进对局。
四狼裸坐
四名狼人不做深水或倒钩，集体明确站边、冲票并强行抢轮次的打法。
对话
在自己的发言中明确把某段话说给指定玩家听，对方仍需等到自己的发言顺序才能回应。
带走
通过猎人开枪、女巫毒药、白狼王自爆等技能使另一名玩家出局。
开枪
猎人或具备枪杀能力的身份出局时发动技能，选择一名玩家一同出局。
颜杀
在缺少有效发言信息时，仅根据表情和即时反应判断身份。
打煽动
用强势发言推动其他玩家支持某个站边或归票目标。判断时要看其逻辑是否完整，而不只看语气。
推
通过白天投票让一名玩家出局；狼队夜间手势中也可能表示某张牌适合留到白天抗推。
抿爆
持续准确施压并点出狼人身份，使对方因难以继续隐藏而选择自爆。
爆刀
狼人自爆后按房规指定当晚刀口；对白狼王也可指自爆直接带走目标。
强神
技能影响轮次较大的神职，常指女巫、猎人等，具体范围随版型变化。
弱神
技能较被动或自证能力较弱的神职，常见例子是白痴。
枪
猎人的口语简称，也可泛指拥有出局开枪能力的身份。
推不出去的牌
通常指白痴等被放逐后仍可能翻牌留场的身份，具体以版型规则为准。
认刀认推不认毒
常用于猎人、狼王或白痴说明自己的出局技能：被刀或被推时能发动，被毒时不能发动；实际效果以身份规则为准。
蛋
平民牌的口语称呼。
先知
预言家的另一种称呼。
爱神
丘比特的另一种称呼。
混混
混血儿的口语简称。
浅水狼
已经进入焦点、发言或状态有明显问题，但仍有机会被留在场上的狼人。
悍跳狼
起跳预言家或其他关键神职，与真身份正面对抗的狼人。
山洞狼
隐藏较深但仍保持一定发言存在感、持续参与分析的狼人，含义接近深水狼。
金宝宝
双身份或选牌规则中，玩家拿到两张平民牌的口语说法。
身份做好 / 做差
一名玩家的发言、行为或票型让其更像好人，或更像狼人。
身份高 / 低
身份越高表示越可能是好人，身份越低表示越可能是狼人；高低是当前判断，不等于底牌事实。
愚民
没有看清局势、投错关键票的村民。该词带有贬义，交流时宜少用。
乱民 / 暴民
因发言或操作搅乱好人视野的村民。该词带有贬义，应优先讨论具体行为。
民及民以下
身份可能是平民或狼人，意思是发言没有达到神职应有的信息量或视角。
铁好人
根据已确认规则和信息，判断某名玩家必定属于好人阵营。
隐狼
泛指长期隐藏身份的狼人；在某些版型中也可能是一个有独立技能的正式身份。
第三方
好人和狼人之外、拥有独立胜利条件的阵营，例如部分规则下的人狼情侣。
链子
丘比特指定成为情侣的两名玩家；其中一方出局时，另一方通常会殉情。
榜样
野孩子、混血儿等身份首夜选择并跟随的目标；榜样如何影响阵营或技能以具体版型为准。
过
表示自己的发言结束，请下一位玩家继续发言。
枪走
猎人发动开枪技能把某名玩家带走。
插话
尚未轮到自己发言时打断其他玩家。多数面杀规则不允许随意插话。
